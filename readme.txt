Area calculator to find the surface area of the following shapes - rectangle, triangle, circle, sector, ellipse, trapezoid, and parallelogram.

[https://www.calculator.io/area-calculator/](https://www.calculator.io/area-calculator/)

== Usage ==

Add the Area Calculator shortcode to your page, post or sidebar:

`[ci_area_calculator]`

Add the following code to your template where you would like the Area Calculator to appear:

`<?php display_ci_area_calculator(); ?>`

== Installation ==

1. Upload the Area Calculator /ci_area_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Area Calculator plugin through the "Plugins" menu in WordPress.
